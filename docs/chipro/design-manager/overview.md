# Design Manager Overview

Design Manager is the shared home base inside Chipro. It keeps projects, libraries, categories, and cells aligned in one screen while tool shortcuts, automation hooks, and AI co-pilots stay in sync.

## Workspace Snapshot

Projects, libraries, and categories sit in dockable sidebars around the workspace.
The central grid mirrors your selection instantly, showing cells with status chips.
A right rail keeps cell metadata, timestamps, and quick actions one tap away.
Layout presets stick to your profile so preferred panels remain pinned on reload.
