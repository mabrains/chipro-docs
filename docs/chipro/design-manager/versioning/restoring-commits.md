You can restore your project to any previous commit at any time.

![Version List](@site/static/img/chipro/designManager/versioning/VersionsList.png)

Hover the commit you want to return to, then click **Restore**.

![Restore](@site/static/img/chipro/designManager/versioning/VersioningBase.png)

Your project files will be reverted to the selected commit.
