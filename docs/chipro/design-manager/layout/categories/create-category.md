To create a new category:

1. Select a **library**.
2. Open the **Categories** panel.
3. Click the **“+”** button.
4. Enter a **Category Name** and optional **Description**.
5. Click **Create Category**.

![Create Category](@site/static/img/chipro/designManager/category/CategoryCreate.png)

After submitting, the new category appears in the list immediately.

![Category Added](@site/static/img/chipro/designManager/category/CategoryCreated.png)
