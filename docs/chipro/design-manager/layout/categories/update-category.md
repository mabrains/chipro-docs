You can update a category’s name or description at any time.

1. Open the category options menu (⋯).
2. Click **Update**.
3. Change the **Category Name** or **Description**.
4. Click **Update Category**.

![Update Category](@site/static/img/chipro/designManager/category/CategoryUpdate.png)

Changes are applied immediately and reflected across the workspace.
