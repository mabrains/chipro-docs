You can change a cell’s category at any time.

1. Open the cell’s **three-dot menu**.
2. Select **Change Category**.

![Change Category Dialog](@site/static/img/chipro/designManager/cell/ChangeCategory.png)

3. Choose a category, or select **No category** to keep it uncategorized.
4. Use the search bar to filter categories.

If no categories match your search:

![No Categories Found](@site/static/img/chipro/designManager/cell/ChangeCategoryNoResults.png)

5. Click **Save changes** to apply the update.
