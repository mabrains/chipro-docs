---
title: Delete a Project
sidebar_label: Delete
---

Deleting a project permanently removes its data, including libraries, cells, and connections.

---

## Opening the Delete Dialog

Select **Delete** from the project options menu.

![Project Options](@site/static/img/chipro/designManager/project/ProjectOptions.png)

---

## Delete Confirmation

Chipro displays a clear warning outlining what will be removed:

- Project configuration
- Associated libraries
- Cell data and references

![Delete Project Dialog](@site/static/img/chipro/designManager/project/DeleteProject.png)

Once confirmed, the deletion **cannot be undone**.

If all projects are deleted, Chipro returns to an empty workspace state.
