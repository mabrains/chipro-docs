The **Actions** panel provides quick access to global project operations such as installing Process Design Kits (PDKs) and managing version history.It is located at the top-right area of the Design Manager workspace.

![Actions Panel](@site/static/img/chipro/designManager/actions/ActionsExpanded.png)

---

## Minimize & Expand

You can minimize or expand the Actions panel using the toggle button centered at the top of the panel.

This is helpful when you want more vertical space for browsing cells.

![Actions Minimized](@site/static/img/chipro/designManager/actions/ActionsCollapsed.png)

---

## Add PDK

The **Add PDK** button opens the PDK installation dialog where you can:

- Browse available PDKs
- Install, reinstall, or remove PDKs
- View details such as technology, vendor, and version

This feature is documented in detail in:
[Process Design Kits (PDKs)](../pdks/overview.md)

![Add PDK Modal](@site/static/img/chipro/designManager/actions/OpenPdks.png)

---

## Version History

The **Version History** button allows you to:

- Create commits
- Track project-level design changes
- View or manage previous versions

For full details, refer to:
[Version History](../versioning/overview.md)

![Version History Modal](@site/static/img/chipro/designManager/actions/OpenVersioning.png)
