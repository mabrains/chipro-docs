You can create a new library for any active project.

---

## Empty State

When no libraries exist, Chipro shows a simple prompt to help you begin.

![Empty Library State](@site/static/img/chipro/designManager/library/LibraryEmpty.png)

---

## Opening the Library Form

Click **Create Library** or use the **“+” button** in the Libraries panel.

![Create Library Dialog](@site/static/img/chipro/designManager/library/CreateLibraryEmpty.png)

---

## Filling Library Details

Enter the required information:

- **Library Name**
- **Library Description** (optional)

Example of a filled form:

![Library Filled](@site/static/img/chipro/designManager/library/CreateLibrary.png)

---

## Library Created

After submission, the new library appears immediately in the Libraries panel.

![Library Added](@site/static/img/chipro/designManager/library/LibraryAdded.png)
