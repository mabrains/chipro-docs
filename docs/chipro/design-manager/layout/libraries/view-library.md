You can view detailed information about any library.

---

## Opening Library Details

Open the library menu and click **View**.

![Library Options](@site/static/img/chipro/designManager/library/LibraryOptions.png)

---

## Library Details Dialog

The dialog shows:

- Library name  
- Number of cells  
- Description  
- Creation and last update timestamps  

![View Library](@site/static/img/chipro/designManager/library/ViewLibrary.png)

Use this to quickly inspect metadata of the library.
