## Options Menu

Click the three-dot icon next to any library, or right-click the project card to open its actions.

![Library Options](@site/static/img/chipro/designManager/library/LibraryOptions.png)

Available actions:

- **View** — Inspect library information  
- **Update** — Edit the library name or description  
- **Delete** — Permanently remove the library  

These actions help you manage your hierarchy efficiently.
