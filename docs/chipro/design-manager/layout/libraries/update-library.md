You can update a library’s name or description at any time.

---

## Opening the Update Form

Click **Update** from the library’s options menu.

![Library Options](@site/static/img/chipro/designManager/library/LibraryOptions.png)

---

## Editing Library Information

Modify the fields as needed:

- Library name  
- Description  

Example:

![Update Library](@site/static/img/chipro/designManager/library/UpdateLibrary.png)

After saving, the updated information appears instantly in the Libraries panel.
