title: Delete a Library
sidebar_label: Delete
---

Deleting a library permanently removes:

- All cells within the library
- All cell configurations and data
- All connections to other libraries
- Library metadata and settings

---

## Opening the Delete Dialog

Select **Delete** from the libraryâ€™s options menu.

![Library Options](@site/static/img/chipro/designManager/library/LibraryOptions.png)

---

## Delete Confirmation

A warning dialog appears summarizing everything that will be removed.

![Delete Library](@site/static/img/chipro/designManager/library/DeleteLibrary.png)

This action **cannot be undone**.  
Once confirmed, the library and all related data are permanently deleted.
