To install a PDK, open the PDK Manager and select any PDK from the **Available** tab.

Click **Install** to begin.

![Available PDKs](/img/chipro/designManager/pdk/PDKs.png)

---

## Installation Process

During installation, Chipro will:

1. Download the PDK archive
2. Extract and verify files
3. Build tool environments (Xschem, KLayout, etc.)
4. Activate the environment

A live installation log is displayed:

![Installing](/img/chipro/designManager/pdk/Installing.png)

You can **Copy** the logs or **Open** them in a full-screen view.

![Installing Full](/img/chipro/designManager/pdk/FullMode.png)

---

## Installation Complete

When finished, you’ll see:

![Complete](/img/chipro/designManager/pdk/Complete.png)

After installation:

- The PDK becomes available in the **Installed** tab
- You can assign it to new projects
- Dependent views (schematic, layout, symbol, netlist) become fully functional

**Button Behavior:**
- **Continue** — takes you directly to the **PDK Manager**  
- **Close** — simply dismisses the dialog
