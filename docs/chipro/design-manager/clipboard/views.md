# Copy or Move Views

Use this guide to handle view-specific clipboard actions, from selecting views to completing a paste inside the destination cell.

---

## 1. Select Views

Choose a cell, open the **Views** panel, and activate selection mode.

![Selecting Views](@site/static/img/chipro/designManager/clipboardSystem/view_1.png)

You can select multiple views at once.

---

## 2. Copy or Move Views

With views selected, click **Copy** or **Move** in the toolbar.

![View Actions](@site/static/img/chipro/designManager/clipboardSystem/view_2.png)


---

## 3. Paste Views into Another Cell

Navigate to the target cell. If it accepts the clipboard contents, a **Paste** button appears.

Open the dialog to review the transfer.

![Paste Dialog for Views](@site/static/img/chipro/designManager/clipboardSystem/view_4.png)

The dialog displays:

- Source cell.  
- Target cell.  
- Views being copied or moved.  
- Any extra configuration.

Click **Paste** to finalize once the summary looks correct.

---
