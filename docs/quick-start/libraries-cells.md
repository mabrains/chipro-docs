# Add libraries and cells

1. Select your project and  then [Create Library](../chipro/design-manager/layout/libraries/create-library.md).
2. Select your library and then [Create Cell](../chipro/design-manager/layout/cells//create-cell.md).
3. Select your Cell and then [Create View](../chipro/design-manager/layout/views/create-view.md).
4. Now select your View and open it in Vnc
